## Temperature Monitor System

A Python-based temperature monitoring system simulating Arduino sensor data.
Built as part of my Smart Manufacturing engineering studies at IIITDM Jabalpur.

## What it does
- Simulates 20 temperature readings from a sensor
- Detects overheating above 40°C automatically
- Visualizes readings with color coding (green = Safe, red = Overheating)
- Logs all readings to CSV for record keeping

## Technologies Used
- Python
- Pandas
- Matplotlib
- Random (sensor simulation)
